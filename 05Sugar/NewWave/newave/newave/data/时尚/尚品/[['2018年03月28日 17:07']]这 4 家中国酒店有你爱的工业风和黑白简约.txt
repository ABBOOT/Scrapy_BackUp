　　导语：这年头，一个产品不拿一个红点、一个 iF，可能都不好意思说自己有设计感。（来源：太平洋时尚网）
　　iF 设计大奖昨天揭晓，5 家酒店中有 4 间在中国！这些就是 iF 评委眼中的设计酒店。
　　就在昨天，iF 的官网新鲜出炉了2018年度建筑大奖，评选出过去一年的最佳设计。
　　iF 的知名度已经到了无需多言的地步。
　　这年头，一个产品不拿一个红点、一个 iF，可能都不好意思说自己有设计感。
　　于酒店而言 iF 大奖相对冷门，却能带来全新的思路。
　　去年大奖公布后，他就给 MISS 挖到了一个把建筑和自然融合的酒店。
　　在今年，获得 iF design Award 2018 的中国酒店，就有 4 家之多，几乎把本来就不多的酒店分类给包场了。
　　其中有如洲际这样的大牌酒店，更多的是独立的设计型酒店。
　　有的是简约的工业风，还有地试图阐释艺术与度假的关系。
　　
　　位于台北市中心，尚未开业的 Light Hotel 是台湾‘境觀設計’的全新作品。
　　这是一个极其特殊的酒店，内部走了灰暗的工业风格。
　　酒店的大堂，乃至于通道均以未经掩饰的水泥组成。
　　乍一看，整个酒店将工业社会中最原始的材料，拼接出了美感。
　　室内并没有延续公共区域里冰冷的样子，反而用暖色原木和灯具增加温暖的感觉。
　　酒店尚未开业，就连连获奖。
　　除了刚刚颁发的 iF 设计大奖外，还拿了日本 Good design Award 的优良设计奖。
　　iF 评委对 Light Hotel 中，植根于自然的审美特征十分赞赏。这也是酒店获奖的理由。
　　
　　西海艺术湾酒店这是一个位于青岛，尚未开业的酒店。
　　从大堂到客房，在西海艺术湾的环境中，艺术故事与内部装饰融合。
　　酒店大量采用落地玻璃的设计，让自然光进入室内。
　　泳池则面对唐岛湾，空间极简。
　　西海艺术湾酒店共有 69 个房间，室内也是延续了公共区域的设计，又有绿植带来的度假风。
　　整个酒店最为特殊的一点在于：房间层高 5.25米。
　　这就意味着除了常设的艺术品，酒店还能够开设艺术特展。
　　同样位于西海艺术湾的青年旅舍，也获得了 iF 建筑类大奖。
　　iF 大奖在获奖介绍中，特别提出西海艺术湾是一个‘艺术家酒店’，能让建筑和艺术美好相遇。
　　
　　酒店位于三亚湾椰梦长廊海边，用高级灰打造了一种全新的三亚度假风格。
　　在大海的包围下，酒店没有采用海滩度假的风格，反而是在低调又活泼的黄色中配有简约灰白。
　　房间内的设计通透，卫浴空间与起居室用一个可以移动的门隔开。
　　床头又十分温馨，简约灰色格调，创造真实性独特性，实现对空间的想象力。
　　这座酒店没有过分强调自己的设计感，却通过琐碎的饰品、立面的细节体现出艺术的氛围。
　　在设计上，iF 的评委认为，酒店融合了传统奢华与中国传统价值观，能让人在海边享受到田园诗篇。
　　
　　这是整个 iF 设计大奖中，唯一一家国际连号酒店。
　　酒店从大堂开始，以宝石蓝作为主色调，使整个空间拥有前卫的科技感。
　　在公共区域内，还呈现出六角形的钻石元素。
　　红、蓝的亮色碰撞，使得整个室内显得十分年轻。
　　客房中保持了宝石蓝的色调，起居与休息的空间间隔分明。
　　对钻石的六角形结构的解构和‘雅痞’风格的体现，是酒店获得 iF 大奖的原因。
　　
　　这是酒店设计分类中，唯一的一家海外酒店。
　　Hotel Mono 位于新加坡唐人街，讲述了一个曾经的经济型酒店，翻新成现在黑白结合的设计酒店的故事。
　　酒店的前台并不大，却用简单的几何效果做出设计感。
　　由于酒店是由唐人街的老建筑翻修而来，酒店拥有 7 个不同房型。
　　室内全都以黑白为主色调，真个环境简单到让人感觉是一个艺术品。
　　不过窗户用的倒是洛可可时代的样式，给本来现代设计风格添上一丝人文气息。
　　酒店已经开启了预定，价格约为 600 元/晚起。
　　iF 的对这处酒店的介绍说：该项目将曾经的经济型酒店，爆改拥有46间客房的别致天堂，散发出简约而不失自信的设计语言。
