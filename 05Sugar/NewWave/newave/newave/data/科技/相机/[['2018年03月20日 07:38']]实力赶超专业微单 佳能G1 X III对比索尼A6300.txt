　　以前，如果你想学习摄影，有经验的人士一定会给你推荐一堆专业又沉重的摄影器材。而随着轻摄影的逐渐流行。无论是专业摄影师选择轻便的“第二台相机”还是摄影爱好者购买自己的入门设备，需求除了成像素质高之外，还希望尽可能的轻便，预算最好控制在5-8千。目前市面上的相机，能称得上轻便的，除了微单就是卡片相机了，但是以往卡片相机兼顾高素质成像效果的产品又很少，但是佳能前不久发布的G1 X Mark III就是两者兼顾的一款产品。那么这款搭载了2420万有效像素APS-C画幅传感器设计的产品，能否取代微单成为摄影爱好者或是专业摄影师随行拍摄的明智之选？带着这个问题，我们找来了市面上这个价位段的热门产品索尼A6300作对比，从实用角度与大家探讨，究竟谁更合购买。
　　● 
　　为了公平起见，为索尼A6300选用了索尼E PZ 16-50mm f/3.5-5.6 OSS这颗套机头，等效焦段为24-75mm，与佳能G1 X III等效24-72mm的焦段基本保持一致。从图中我们可以看到，两款相机虽然都是APS-C画幅的产品，具有专业的摄影能力，但是体积上相差较多。虽然机身厚度方面，两款产品相差较小；但是如果与加了镜头的索尼A6300相对比，佳能G1 X Mark III的优势就十分明显了。索尼A6300镜头的体积接近于佳能的2-3倍，带来的直接影响是，我可以将佳能G1 X Mark III放入衣裤口袋随身出行，而携带索尼A6300就需要背包了。
　　● 
　　索尼A6300和佳能G1 X Mark III都设计了翻转屏，但是索尼的翻转屏角度很小，无法完成自拍等特殊视角拍摄操作。而佳能G1 X Mark III能够支持水平180度和竖直270度的翻转，几乎是想怎么拍就怎么拍。除了翻转屏的角度比索尼A6300有明显优势之外，佳能G1 X Mark III的屏幕还支持触控操作，调节参数、设置选项，甚至是对焦、拍照、录视频都可以通过触摸屏幕实现，让用户使用起来更简易。不仅如此，佳能G1 X Mark III还支持使用取景器拍摄触摸和拖拽显示屏自动对焦，让摄影师可以更好的集中精力捕捉被摄体。而使用索尼A6300拍摄则需要多个组合按键共同调节完成，这在进行运动物体抓拍及场景切换时，往往会因为过程繁琐而感到力不从心。
　　● 
　　既然两款产品都是APS-C画幅，那么自然就要针对画质方面进行一些对比了。话不多说，我们直接上对比照片（
）。
　　
　　在光线良好的情况下，我们可以看到两款相机不论是画质、锐度方面都比较接近，并且照片都没有明显的瑕疵。整体直出照片的质量，我们认为都是合格的，且难分高下。
　　
　　在暗光环境下，佳能G1 X Mark III的直出照片色温和白平衡都很准确，与真实观感无异。而索尼A6300的直出照片可以看到明显的偏色，画面整体偏紫，无论楼、天空和船身的颜色都或多或少的受到了影响。
　　
　　在夜间，我们不仅可以提升高感手持拍摄，也可以放在三脚架上进行低感光度的长曝拍摄。在相机参数、镜头焦段均一致前提下。100%放大截图，可以看到非常明显的细节差异。
　　从放大的对比截图我们可以看到，佳能G1 X Mark III可以清晰的记录下建筑的所有细节，大部分建筑上的文字和细节也记录的非常完整。相比而言，索尼A6300呈现的放大区域中，画面的细节丢失比较明显，尤其是楼体的边缘、砖缝、窗户等等细节都已经模糊了，与佳能G1 X Mark III相比，差异明显。
　　● 
　　通常来说卡片相机的画幅较小，控噪能力并不会十分出色。而佳能G1 X III则不同，它搭载了支持全像素双核对焦的APS-C画幅图像感应器，结合DIGIC 7影像处理器，可以实现高画质的照片拍摄。
　　从上面两张照片就可以明显的看出佳能G1 X III相机控噪能力和夜间画质效果都非常出色。首先，画面十分干净，即便是夕阳西下亮度明显不足的情况下，整个画面的噪点依然非常少。尤其是在暗光的区域，比如平房的房顶和SOHO楼宇间的街道。
　　为了体验佳能G1 X III的控噪效果，我们拍摄了更昏暗的环境。一个是太阳落山后北京国贸的经典机位，另一个是和风饭店的室内场景。我们看到在更昏暗的环境下，佳能G1 X III相机的控噪效果依旧很出色，比如国贸楼群的昏暗处，再比如饭店室内场景左侧的“试营业”牌子，都很难看到明显的噪点，充分展现了产品在控噪部分的实力。
　　● 
　　我们可以看到上图中，在不同光线和色温的状况下，佳能G1 X III相机都能够精准的还原色彩。不论是夕阳照射下的地面还是紫色的天空，相机记录的色彩与我们肉眼看到的色彩没有差异。
　　总体来看，两款产品均配备了APS-C画幅的CMOS处理器，而佳能G1 X Mark III是卡片相机，索尼A6300是微单相机，体积相差十分明显，包括镜头的体积。我们日常出门的时候，佳能G1 X Mark III机身大小和手机无异，可以轻松放入口袋，而索尼A6300根本无法实现。另外，佳能G1 X Mark III不仅体积小，还可以胜任很多种场景和环境的拍摄。不论是成像素质还是操控性能都足够专业，还设计了更出色且实用的触摸翻转屏。考虑到索尼A6300要充分实现其性能还需要购置不少镜头，又是一笔不小的开销。因此推荐喜欢轻装出行的摄影师及爱好者们选择佳能G1 X Mark III。
