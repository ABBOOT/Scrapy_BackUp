　　现在人们出门旅行，对摄影装备的便携性要求越来越苛刻，因此很多人干脆就直接选择卡片相机。毕竟小巧便携无需背负沉重摄影器材，受到很多轻旅行爱好者追捧和喜爱。当然卡片相机发展至今，已经今非昔比了，现在市面上脍炙人口的产品一抓一大把，比如佳能G7 X Mark II和索尼RX100 V（以下简称黑卡5）。两款产品单从定位和参数上来看比较接近，因此很多有需求的用户经常会在它们之间犹豫不决，纠结该选谁？下面通过几组对比看一下佳能G7 X Mark II和索尼RX100 V谁的优势更明显。
　　
　　首先，我们先来进行直出照片的对比。本组对比中，我们采用两款相机在完全相同的环境直接拍摄同一场景，然后对比两款相机的JPG直出照片。
　　通过对比我们可以看到，佳能G7 X II的直出照片色彩还原非常准确，不论是鱼肉的颜色还是桌面的颜色都可以完美的还原。并且拍摄环境是黄色灯光，佳能G7 X II的色温也没有受到灯光的过多影响。但是索尼黑卡5的偏色问题比较严重，鱼肉已经完全不是肉眼看到的色彩。在画面的细节方面，佳能G7 X II也比索尼黑卡5稍好一些，比如鱼肉的纹理和木盆的边缘等等细小之处，佳能的保留更加完整。
　　
　　对于卡片相机来说，虽然现在得到了显著地性能和画质提升，但是一般来说高ISO的控噪表现仍然是短板。为了呈现两款相机的控噪能力，我们特地选择了夜间进行对比。ISO方面选择了3200，对于卡片相机来说，这个ISO应该是比较极限的范围了。
　　中央区域，我们可以明显的看到佳能G7 X II的控噪效果好于索尼黑卡5。不论是楼面还是下面“观光船”的文字牌，都可以看到索尼黑卡5的噪点明显更高。在细节的部分，佳能G7 X II的细节保留也更加完整。另外，我们可以看到，在ISO 3200这个档位，佳能G7 X II完全可用，而索尼黑卡5在 ISO 1600之后画质下降明显，且噪点多到不可用。
　　我们再来看看边缘区域。与上面类似，佳能G7 X II的控噪能力依旧比索尼黑卡5要出色。我们可以看到画面中楼体的外表处，索尼黑卡5的噪点明显更多，很多砖缝的细节之处，索尼黑卡5已经丢失，佳能G7 X II还可以清晰地呈现。另外楼体中灯光的效果方面，佳能G7 X II也同样的保留了完整的细节。
　　● 
　　佳能G7 X Mark II和索尼黑卡5不论是外观大小，还是重量方面，都十分接近，而且配色都是黑色，就连按键的布局方面，两者都非常相似。
　　其实很多人在使用相机的时候，并不适应按键或者组合按键调节，特别是一些非专业摄影师，经常因为调节参数繁琐而感觉到混乱。为此，佳能G7 X Mark II贴心的设计了触摸屏幕。这样，不论你是习惯于触摸操作还是入门用户，都可以非常快速的上手使用。另外，佳能G7 X Mark II的触摸屏幕不仅可以支持触摸调节参数和设置，还支持触摸对焦和触摸快门，也就是我们在日常拍摄的时候，都可以通过触摸直接完成。而使用索尼黑卡5拍摄需要组合按键共同调节完成，这在抓拍一些关键场景时，往往会感到力不从心。
　　由于卡片相机较小，尤其是对于男性用户来说，机身的大小比手掌要小。因此握持手感也对拍照稳定性有很大的影响。佳能G7 X Mark II贴心的设计了手柄，在握持的时候可以和手掌更好的贴合，不容易滑落造成磕碰等损失。而索尼黑卡5则没有任何握持的设计，用户需要自己单独购买相关配件。从这一点来看，佳能G7 X Mark II的握持舒适性明显好于索尼黑卡5。
　　● 
　　综合上面的对比，我们可以看到佳能G7 X Mark II在多方位的表现上都比索尼黑卡5更加优秀。那么接着，我们再来看看我们在日常出行时，佳能G7 X Mark II的实际表现如何。
　　我们从照片中就可以看到，佳能G7 X Mark II的直出照片画质非常出色，水面波纹及高楼窗户细节清晰可见。同时色彩还原也十分精准，远处的红墙及侧面的护栏并没有因为阳光直射而失去其本来的色彩，与真实观感无异。
　　而对于光线比较昏暗的场景，佳能凭借着DIGIC 7处理器的优势，即使场景光线不足的情况下，也能够准确还原每一种颜色。
　　总体来说，我们可以看到不论是直出照片的色彩还是控噪方面的表现，佳能G7 X Mark II都要比索尼黑卡5更优秀。手柄的设计增加了握持的舒适性，触摸屏幕设计简化了操作的难易程度，Wi-Fi和蓝牙的配备让用户可以更加便捷的将照片传输至手机，进行及时的分享。除此之外，不得不说的是索尼黑卡5在京东商城售价约6000元，而佳能G7 X Mark II售价不到4000元。如果说在这两款类似规格定位的相机中选择，笔者更倾向于比较全能且售价更实惠的佳能G7 X Mark II。
