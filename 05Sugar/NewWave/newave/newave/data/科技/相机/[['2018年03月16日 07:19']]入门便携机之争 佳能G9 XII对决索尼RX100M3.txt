　　冬去春来，想必很多用户都有外出游玩的想法，这时一英寸便携机型就是个不错的选择，既能提供不俗的性能表现，同时减轻了旅途的行李负担，此外小巧的机身对入门新手而言也更加友好。
　　说到一英寸便携机，佳能G9 X Mark II、索尼RX100M3是目前很多摄影初学者关注的焦点，性能、体积方面它们均有着不错的规格，而且长期以来佳能、索尼两个厂商在便携机型市场也都颇有建树。那么这两者的便携性与拍摄体验分别如何？谁更适合入门用户使用呢？笔者本次就为大家带来对比。
　　
　　首先来说说外观。佳能G9 X Mark II提供了银色、黑色两种版本，此次我们拿到的是银色版，银色机身加上棕色蒙皮，让这款小相机颇具复古文艺的气质；另外在机身线条上，G9 X Mark II的造型颇为硬朗，加上复古文艺的配色，整体颜值非常抢眼。
　　与佳能G9 X Mark II不同，索尼RX100M3在外观上走的是极简路线，整个机身均采用全黑配色，同时在造型上使用了两侧圆润、前后硬朗的机身线条，整体强调了索尼一贯的“黑科技”风格。不难发现这两款相机的设计风格截然不同，佳能更具时尚色彩，索尼相对低调朴实。
　　看过颜值，接着就要看看便携性了。先说整体重量（包含电池及存储卡），佳能G9 X Mark II重量约为206克，索尼RX100M3重量约为290克，两款卡片机的重量都控制在了300克以内。虽然数字差异不大，但同时手持会发现索尼RX100M3要明显更沉一些，如果是单手握持拍摄，佳能G9 X Mark II的体验要更加轻松。
　　再看机身体积。同为便携机型，这两者的机身尺寸还是有区别的，为了方便比较，笔者将两款机型放在一起进行对比。可以看到在大小、厚度以及镜头突起高度上，佳能G9 X Mark II都要更为小巧一些，综合两款机型的重量数据，不得不说佳能G9 X Mark II在便携方面做得更加极致。
　　
　　外观对比之后，接下来看看这两款相机的上手体验分别如何。佳能G9 X Mark II与索尼RX100M3的上手感受可以说截然不同，不仅是重量差异，握持感上也有着不一样的体验。佳能G9 X Mark II在机身双面均配有蒙皮材质，握起来更舒适，同时可以防止机身滑脱，这一设计还是很贴心的。
　　索尼RX100M3在握持手的拇指处提供了一小块蒙皮，用以改善抓握体验，但在机身正面并没有蒙皮或手柄设计，因此这款相机整体手感较为光滑，单手使用容易出现手滑的问题，笔者个人建议搭配挂绳或是加装手柄来避免滑脱。
　　除了手感之外，相机操控性也决定了整体的使用体验。先来说说按键设计，佳能G9 X Mark II的键位非常简洁，进一步降低了使用门槛，即便入门用户也可以轻松使用；索尼RX100M3的按键较为复杂，对于专业用户而言这无疑是件好事，但摄影初学者在刚上手时需要熟悉一段时间。
　　再来看看实际操控体验。为了让大家看得更直观，笔者用视频形式来进行展示。作为一款极致轻便的相机，G9 X Mark II主打触控操作，通过触屏来完成参数设置、图像翻页查看/管理等操作，整体使用体验与智能手机相似，非常简单；索尼RX100M3采用纯按键的操控方式，实拍中通过按键+拨轮的形式完成调整，这样的设计更适合专业玩家，不过对入门用户而言显得不够友好。
　　总的来说佳能G9 X Mark II与索尼RX100M3作为两款便携口袋机型，都提供了不俗的操控表现，但在操控逻辑上走了两条完全不同的路线。就笔者实际使用的感受而言，索尼RX100M3针对专业摄影玩家做了一系列优化，上手门槛较高；佳能G9 X Mark II则主打触屏，更适合入门新手使用。
　　
　　最后我们来对比下这两款相机的性能表现，作为主打便携的口袋机型，两款相机都配备了2000万像素级别的1英寸传感器，成像方面的核心参数基本一致。我们先来看看这两款相机的实拍成像效果如何——
　　从实拍照片来看，这两款相机都能带来不俗的成片质感，但进行100%放大后则有一定差异，佳能G9 X Mark II的成像质量更加扎实，画面细节部分更清晰；索尼RX100M3的细节部分有些发虚，整体成像质量要逊色一些。
　　不只是细节，相机的直出色彩同样是评判其成像能力的重要标准。从上面一系列对比样张中，不难发现佳能G9 X Mark II提供了更鲜艳、饱满的直出色彩，索尼RX100M3的直出风格相对比较平淡、偏冷。色彩这个东西比较主观，具体喜欢哪家的颜色还得看用户自己的喜好。
　　除了成像之外，相机的拍摄效率也是很多入门用户关注的焦点，这时对焦速度就显得尤为重要了。我们使用这两款相机在同场景下进行了实拍对比，并且通过视频形式来向各位读者进行展示，看看谁的对焦速度更胜一筹。
　　从实际拍摄的体验来看，佳能G9 X Mark II具备更快的对焦速度，同时借助触屏操作和触摸快门功能，可以轻松实现点哪拍哪，因此在实际使用中能够提供更出色的对焦效率；索尼RX100M3通过按键来选择对焦点，并且在合焦时偶尔会出现拉风箱的问题，因此在对焦体验上要慢了不少。
　　
　　经过外观、上手体验、实拍表现的对比，相信各位对这两款相机已经有了比较全面的认识。既然选择带小相机出门，那么轻巧、便携当然是第一要素，便携性这方面，佳能G9 X Mark II提供了更小更轻的机身，随身携带、单手拍摄要更加小巧轻便；另外，G9 X Mark II在颜值上有着更抢眼的表现，从机身设计而言，佳能这款产品无疑更有竞争力。
　　使用体验方面，两台相机展现了不同的设计理念，佳能G9 X Mark II主打触控操作，触屏体验对于入门用户而言更加友好；索尼RX100M3则将自家微单的操控习惯塞进了这款小卡片，专业用户或许能够较快上手，但对新手而言则需要熟悉一段时间，这一点也是各位有购机需求的读者所需要注意的。
　　最后要说说价格因素，目前佳能G9 X Mark II售价为2899元，索尼RX100M3售价为4099元（均为京东品牌官方店的价格）。对于各位摄影初学者而言，如果你准备入手一款极致便携的拍摄装备，同时希望自己的相机能够好看且易用，那么佳能G9 X Mark II无疑是更具性价比的选择。
