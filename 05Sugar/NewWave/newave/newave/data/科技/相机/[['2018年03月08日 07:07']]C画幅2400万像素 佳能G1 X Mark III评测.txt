　　本文来自蜂鸟网
　　如果对相机稍有了解的朋友都知道，佳能G系列一直在佳能数码相机中占据了非常重要的地位，从早年经典的G3、G5，得到后来改为旁轴外观的G9、G11，一直延续着佳能G系列便携专业高画质的特点。在2012年推出的G1 X则是G系列相机1.5尺寸传感器改变的开始，随后的G7 X、G5 X、G9 X等都采用了一英尺传感器，逐步将G系列升级到了大底传感器相机行列。随后，佳能终于发布采用APS-C尺寸传感器的Powershot G1 X Mark III，为G系列带来了一款画质和机身便携性相当平衡的一款便携相机。
　　众所周知，早期大尺寸传感器与小巧机身、变焦镜头很难做到相对完美的融合。很多采用APS-C或者全画幅传感器的便携相机大都采用了定焦镜头，而具备变焦镜头的相机往往都采用一英尺传感器。佳能G1 X Mark III则是将大尺寸传感器、变焦镜头和小巧机身三者完美结合在一起的产品。
　　佳能官网将G1 X Mark III描述为精致、专业操控和高画质为一体的便携相机，而有些用户则将G1 X Mark III形容为“口袋怪兽”，不论哪种形容，都体现了G1 X Mark III的一个特点，机身小巧、携带方便。佳能G1 X Mark III的机身尺寸为115.0×77.9×51.4mm，仅比采用一英尺传感器的佳能G5 X112.4×76.4×44.2m稍大，重量399g（含电池和储存卡）。在如此小巧的机身内是一枚APS-C画幅、2400万像素传感器，镜头为3倍光学变焦、焦距约为24-72mm，镜头光圈为F2.8-5.6，并且支持IS防抖。相对于前一代佳能G1 X Mark II来说，G1 X Mark III的机身设计和体积可谓是脱胎换骨。
　　拿到佳能G1 X Mark III，咋看起来外观与佳能G5 X十分类似，但细节之处又有许多不同。佳能G1 X Mark III采用了类似单反机身的造型设计风格，高高凸起的军舰部、EVF取景器、双拨轮设计和丰富的操控按键，让佳能G1 X Mark III添加了更多专业色彩。在机线条上，虽然G1 X Mark III造型与G5 X很相似，但整体上G1 X Mark III线条的过度更加精美，尤其是机身顶部“溜肩”设计，不仅让外观多了复古的味道，也不会像G5X那样让军舰部显得十分突兀。
　　佳能G1 X Mark III的机身材质为硬塑料，机身采用了特别喷涂工艺，加上大面积蒙皮覆盖，不仅让G1 X Mark III有着出色的握持感，机身喷涂材质也让G1 X Mark III摸起来有种金属质感。机身的接缝非常严密，翻转屏的排线隐藏在转轴中，提高了翻转屏幕使用寿命，降低了因翻转屏幕造成磨损的风险。
　　佳能G1 X Mark III的机身细节之处无不凸显专业和操控便捷两个特点。G1 X Mark III的按键风格延续了佳能G系列一贯风格，相机拍摄时的所有参数都可以通过快捷菜单和机身背后方向键快速设置，对于老用户来说很容易上手。另外，佳能G1 X Mark III支持Wi-Fi无线连接和NFC功能，可以通过机身侧面无线按钮直接开启，使用起来十分方便。
　　除了按键之外，佳能G1 X Mark III上还配备了前后双拨轮、镜头变焦环方便用户快速变焦和调整拍摄参数，在机身顶部有常见的变焦拨杆、模式盘、曝光补偿转盘、电源开关等，这些拨盘都凸显了佳能G1 X Mark III的专业性。此外，G1 X Mark III的镜头变焦环和机身后拨轮都可以根据用户自身需要和习惯进行自定义设置，这也让相机的操作更加个性化和自由化。
　　此外，佳能G1 X Mark III还内置了一个236万像素的电子取景器，在实际拍摄时这枚取景器的刷新率非常不错，可以带来不俗的显示效果。其次、G1 X Mark III采用了旋转触控屏，不仅便于自拍，触控屏也提升了相机操作的便捷性。最后，佳能G1 X Mark III的军舰部里隐藏一枚机顶闪光灯，方便拍摄时进行补光。
　　佳能G1 X Mark III的电池依然延续了此前G系列常用的NB-13L。之所以未更换新型号电池，首先是受限于G1 X Mark III机身体积限制，其次是NB-13L通用性更强。虽然佳能官方并未给出佳能G1 X Mark III的续航数据，但日常使用拍摄一天问题不大。
　　使用体验方面，G1 X Mark III机身除了大面积的蒙皮外，还包含了突起的握持手柄以及指柄，握持手感非常扎实不易滑脱；搭载的伸缩式镜头在不使用时突起并不多，对便携性的影响基本可以忽略。此外，G1 X Mark III包装里随机还附送了镜头防丢绳以及PowerShot专属相机背带等小配件。
　　说完了佳能G1 X Mark III的外观，接下来看看佳能G1 X Mark III的画质，这也是用户对于这款相机最关注的地方。毕竟如此小的机身和24-72mm F2.8-5.6伸缩式镜头是否能够带来出色的画质，以及这款采用了APS-C画幅的大尺寸传感器的相机高感画质究竟如何？
　　我们先从感光度测试开始。佳能G1 X Mark III采用了APS-C画幅的CMOS传感器，有效像素为2400万像素，并配备了Digic7影像处理器。佳能G1 X Mark III的感光度范围从ISO100起，最高感光度为ISO25600，本次测试我们在室内进行拍摄，逐级调整感光度数值并保持曝光一致、JPEG直出的方式。以下为100%放大截图区域。
　　在JPEG模式下，佳能G1 X Mark III的高感表现还算不错，在ISO 3200以前，画面表现出非常明显的控噪能力，画面锐度的下降并不明显。ISO 3200之后画面中噪点增多，在ISO 6400之后画质随着感光度提升画面细节丢失非常明显。总体来说，佳能G1 X Mark III在ISO 3200以前的画质质量都可以接受。
　　佳能G1 X Mark III搭载了一枚等效焦距为24-72mm F2.8-5.6伸缩镜头，并且具备防抖功能。接下来就来看一看这款镜头成像画质究竟如何。
　　在测试G1 X Mark III镜头分辨率时，选择室外场景进行拍摄，采用了三脚架固定相机，拍摄同一场景并100%截取中心和边角的画质来进行测试。测试是，我们分别对24mm广角和72mm长焦两个焦段的分辨率进行测试，广角端光圈为F2.8-F16，长焦端光圈为F5.6到F16。
　　从佳能G1 X Mark III镜头24mm的分辨率测试来看，镜头在F2.8是中心锐度表现不错，在收缩两档光圈后镜头锐度达到最佳；画面边缘在最大光圈下稍显松散，随着镜头光圈收缩画质边缘画质有明显改善，画质表现比较稳定。
　　在镜头长焦72mm段测试时，F5.6最大光圈下镜头中心和边缘画质表现都非常不错，随着光圈收缩画质下降并不明显，在光圈F16时画面中心和边缘画质明显松散。
　　总体来说，佳能G1 X MarK III镜头镜头分辨率表现非常不错，特别是在收缩光圈之后，中心和边缘成像质量有着非常不错的表现，也完全可以满足日常拍摄需求，不用担心这枚24-72mm的伸缩镜头会影响G1 X Mark III的成像。

　　现在该对佳能G1 X Mark III做个总结了。经过了这么多天的试用，我对佳能G1 X Mark III有了深刻的体会。
　　佳能G1 X Mark III是一款将画质和体积做到完美平衡的产品，APS-C画幅传感器加可伸缩变焦镜头，可以算是一款划时代的大底相机产品。相对于市场上在售的其他APS-C画幅相机产品，佳能G1 X Mark III不仅采用了可变焦镜头，还可以轻松的放入口袋里，这在以前是不敢相信，而我在试用拍样片时，就曾经将G1 X Mark III放到衣服口袋里。
　　佳能G1 X Mark III拥有2400万像素，带来了非常出色的画质表现，从样片中的色彩和画面细节中都得到了很好展现。此外，佳能G1 X Mark III由于采用了APS-C尺寸的传感器，再加上Digic7影像处理器，相机在高感光度画质表现也相对于一英尺产品有了显著提升。
　　对于这台佳能G1 X Mark III，带来惊喜的同时也有些许的不足。镜头广角端最大光圈为F2.8，而长焦端最大光圈为F5.6，最大光圈的不够大带造成了背景虚化的不足；另外，佳能G1 X Mark III的续航表现仍有提升空间。不过，这是一个双向选择题，更大的光圈和更大的电池必然会造成相机体积过于庞大，牺牲的则是便携性。而如今佳能G1 X Mark III的配置应该是体积和性能最好的平衡点。
　　总之，佳能G1 X Mark III继承了佳能G系列的特点，专业的操作和出色的便携，而APS-C画幅、2400万像素传感器和等效24-72的镜头带来了足够出色的拍摄画质和体验。随着技术进步，佳能将相机的便携性和高画质在G1 X Mark III上做了一个结合，未来想必会有更多的产品采用类似方案，给用户带来更好的拍摄体验。
